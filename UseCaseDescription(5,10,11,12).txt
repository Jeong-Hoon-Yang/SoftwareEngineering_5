
Use Case Descriptions(5,10,11,12) B911035_김세환

1. 판매 완료된 상품 내역 조회 기능
Actor Action
① 판매자가 판매 완료된 상품 내역 조회 기능을 선택함.

System Response
② 상품명, 제작회사명, 가격, 판매된 수량, 평균 구매만족도를 상품명의 오름차순으로 정렬하여 화면에 출력함.


2. 상품 구매 내역 조회 및 관리 기능 ( 구매만족도평가 & 환불신청 ) 
Actor Action
① 구매자가 상품 구매 내역 관리 기능을 선택함.
③ 조회된 구매 내역 중에서 하나의 상품에 대해 환불 신청을 함.
⑤ 조회된 구매 내역 중에서 하나의 상품에 대해 구매만족도 평가기능을 선택함.
⑦ 구매만족도 평가 점수를 입력함.

System Response
② 상품명, 제작회사명, 판매자, 가격, 평균 구매만족도, 구매일 을 상품명의 오름차순으로 정렬하여 구매 내역 조회 정보를 화면에 출력.
④ 환불신청이 완료되어 택배사에 수거 요청을 했다는 알림 팝업을 출력함.
⑥ 구매만족도 평가에 대한 팝업창을 띄움.
⑧ 구매자의 평가 점수를 평균 구매만족도에 반영한 구매내역정보를 출력함.

Alternative Courses
- 환불신청시, 시스템은 외부 택배사 시스템(Actor)에 수거 요청을 함.
- 구매자가 4주가 지난 상품에 대해 환불신청을 하면, 환불신청이 불가능하다는 알림 팝업을 띄움.

3. 상품 구매 내역 삭제 기능
Actor Action
① 구매자가 상품 구매 내역 관리 기능을 선택함.
③ 조회된 구매내역 정보로부터 삭제하고자 하는 거래 내역을 선택함.

System Response
② 상품명, 제작회사명, 판매자, 가격, 평균, 구매만족도, 구매일을 상품명의 오름차순으로 정렬하여 구매 내역 조회 정보를 화면에 띄움.
④ 해당하는 거래 내역의 삭제가 반영된 상품 구매 내역 정보를 다시 화면에 출력함.

Alternative Courses
-구매자가 3개월이 지나지 않은 거래내역은 임의로 삭제하려고 하면, 시스템은 삭제가 불가하다는 팝업 알림 띄움.
-6개월이 지난 거래내역이 있으면, 거래내역 삭제 이벤트(Actor)가 삭제 이벤트 발생시킴.

4. 상품 판매 통계 산출 기능
Actor Action
① 판매자가 상품 판매 통계 산출 기능을 선택함.

System Response
② 해당 판매자가 판매한 모든 상품에 대한 총액 및 평균 구매만족도에 대한 정보를  화면에 출력함.

Alternative Courses
-시스템은 매월 말일, 현재까지 판매한 모든 상품에 대한 통계 정보를 이메일로 공지 전송할 수 있도록 이메일시스템(Actor)에 요청.

5. 상품 구매 통계 산출 기능
Actor Action
① 구매자가 상품 구매 통계 산출 기능을 선택함.

System Response
② 해당 구매자가 구매한 모든 상품에 대한 총액 및 평균 구매만족도에 대한 정보를 화면에 출력함.

Alternative Courses
-시스템은 매월 말일, 현재까지 구매한 모든 상품에 대한 통계 정보를 이메일로 공지 전송할 수 있도록 이메일시스템(Actor)에 요청.
